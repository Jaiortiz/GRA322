Waggle Pet Adoption App

GRA 322

Students were given the task of creating an app to fix a problem. "Waggle" was created to aid users in finding the perfect furry friend. Users are able to enter a zipcode and browse thorugh dogs within the area.The user can then "swipe right" on the furry friend of their dreams, from here the user can message with the adoption center/foster home to gain additional information or set up a time to meet. Additonally, adoption centers/foster homes are able to view user profiles via desktop in order to run background checks to ensure that the future pet parent(s) are a good match. This project emphasized understanment of Ui/Ux design across multiple pages with the additon of advanced prototyping. This project was created using Adobe XD.
