Waggle Pet Adoption App

GRA 322

Students were asked to create an app that solves a problem. "Waggle" was created in order to aid users in finding their furry friend. Users are able to input a zipcode and browse through dogs that are avaible at adoption centers or foster homes in that area. The user is able "swipe right" on dogs that they feel may be a perfect fit for them. From here the user is able to browse thorugh the saved pets and message with the adoption center or foster home in order to gain more information about the dog or set up a time to meet. Additonally, the adoption center or foster home is able to view user profile via the desktop app in order to run background checks on users to ensure that they will make for a good forever home. This project emphasized an understandment of Ui/Ux design along with typography and brand identity knowledge. Waggle was created in Adobe Xd with advanced prototyping.
